The ``MyList`` class inherits from ``list`` and adds a ``print_sorted`` method.

Basic usage:
    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Negative numbers:
    >>> neg_list = MyList([-1, -5, 0, 5, 1])
    >>> neg_list.print_sorted()
    [-5, -1, 0, 1, 5]
